This project is to illustarte the basic concepts of python programming. It covers the following topics:
==================================================================================================
     
-> Python Data Types  : Numbers(Integer and Float), Strings, List, Set, Tuple, Dictionary

-> Python Comments : Single line and multi line comments

-> Type Casting

-> Conditional Statement : If, elif , else

-> Loops : for, while

-> Jump and Transfer statement : break , continue

-> Concept of Functions, Modules and  Packages

-> File Handling

-> Lamdba function

-> Object Oriented Programming using python: 
	Classes, Constructor, Objects, Abstraction, Inheritance, Encapsulation, Polymorphism(Overriding and Overloading)

-> Exception Handling ( Try, Except, Else, Finally)

-> *args and **kwargs
